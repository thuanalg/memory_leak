1) Given a specific RAM area. Example:
	#define BUFF_SIZE 2048
	byte buffer[BUFF_SIZE];
	
Design/implement APIs to manage the buffer as below (use C programming language):
	a) byte* allocate(unsigned int size)
		description: to allocate a number of bytes inside the given buffer (similar to C alloc/malloc)
			params:
				+ size: number of bytes to be allocated
			return value:
				+ not null: pointer to begin of allocated bytes
				+ null: can not allocate
	b) void free(byte* p)
		description: to free an allocated area inside the given buffer
			params:
				+ p: pointer to begin of allocated bytes
			return value: none
			
	Test scenario:
		void test()
		{
			byte* a, b, c, d;
			a = allocate(25);
			b = allocate(197);
			free(a);
			c = allocate(1);
			d = allocate(1500);
			free(b);
			free(c);
			free(d);
		}
